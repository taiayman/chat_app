# Chat App

## Features

- google login
- real time messaging
- send images
- emoji picker
- voice recording ui
- ai chat with gemini
- online/offline status
- message history
- shimmer loading states
- auto scroll to bottom
- mobile responsive
- bottom nav on mobile
- back button in chat
- logout

## Stack

- nextjs 16
- nextauth v5
- postgresql supabase
- prisma
- gemini api
- vercel

## Run it

```
npm install
npm run dev
```
